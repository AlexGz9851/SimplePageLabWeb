From C to Wasm

by AlexGz9851

Posted at some point of February 2021.

Utilizar wasm puede ser util para utilizar codigo fuente en lenguajes como C/C++/Rust, etc.

Por esto, haremos un tutorial para simplificar el proceso de instalación.

Instalación de Wasm para Python

Lamentablemente no existe forma de compilar de Python a Wasm. Existen métodos de transformar archivos .py a .js, como es el caso de Transcrypt. Basicamente, una de las razones por las cuales no puede obtenerse archivos .wasm de archivos .py ( entre otras cosas ) es debido a que de forma realista, Python code no puede ser representado como código estaticamente tipado. (Para más info, lea aquí. )

Instalación de Wasm para C

La forma más simple de utilizar wasm en este caso, es compilar tu código C a código asm.js Para esto, sigue las instrucciones:

Instalación de Emscripten

Emscripten es un compilador que traducirá nuestro código de C a asm.js o a .wasm!. Sigue la guía de instalación / uso de emscripten. Como resultado, el compilador ofrece como archivos de salida: a.out.js y a.out.wasm. En este caso utilizaremos el archivo .js!

Tiempo de -C- odear!

Genera tu código fuente! Puedes utilizar algún ejemplo sencillo, como el clásico Hello World! :D

Ya tienes tu proyecto destino?

El archivo a.out.js generado por emscripten puede ser llamado como un script más de js. Recuerda llamarlo con el tag <script>

Compila tu archivo asm.js!

emcc NOMBRE_TU_ARCHIVO.c -s WASM=1 -o NOMBRE_DESEADO.js

Se añade como archivo de salida el archivo.js, para que tambien compile el código C a este lenguaje. (El default es .wasm).

Ejecuta tu proyecto!

Listo! Tu codigo tiene tu libreria de C integrada desde un script de js. Para correr tu proyecto, recuerda utilizar un servidor local!
